# Calculateur d'impots réalisé par YaFou
> Voir la consigne de ce projet [ici](https://github.com/Grafikart/Challenges/tree/master/JS/2-Impot.fr).

## Logiciels requis
- Git
- Makefile
- PHP
- Composer

## Installation
```bash
$ git clone https://github.com/YaFou/Grafikart-challenge-Calculateur-impots
$ cd Grafikart-challenge-Calculateur-impots
$ make
```
Ouvrer votre navigateur sur `https://localhost:8000` !
